Provide a simple widget to show radio wave propagation and solar-terrestrial data from N0NBH.

== Description ==
This widget plugin is directed primarily toward amateur radio enthusiasts
with WordPress blogs. It makes it easy to include banners provided by
[N0NBH](http://www.hamqsl.com/solar.html) on your WordPress blog. There are
several banners available, and they provide different types of
information about HF radio wave propagation and solar-terrestrial data.

== Installation ==
This widget plugin has been tested on WordPress 2.9.2 and should work
on previous 2.x versions as well. PHP 5.x is required.

= Manual Install =

1. Add a directory called 'hf-solar' (without the quotes) to your 
   '/wp-content/plugins/' directory.
1. Upload hfsolar.php to '/wp-content/plugins/hf-solar/'
   directory.
1. Activate the plugin through the 'Manage Plugins' page in WordPress.
1. Choose the size banner you want.
1. Place the widget in the sidebar using the editor for your theme.

= Automatic Install =

1. Download the [hfsolar.zip](http://ns6t.net/qrzsearch/hfsolar.zip) file.
1. Go to the Manage Plugins page.
1. Click the Add New button.
1. Upload the [hfsolar.zip](http://ns6t.net/qrzsearch/hfsolar.zip) file.
1. Active the plugin through the 'Manage Plugins' page.
1. Choose the size banner you want.
1. Place the widget in the sidebar using the editor for your theme.

== Frequently Asked Questions ==

= What is HF propagation and solar-terrestrial data? =

It's a popular ham radio site where hams lookup information about operators
based on their call sign. It also has lots of other ham radio information.

= Am I allowed to do this? =

N0NBH [allows](http://www.hamqsl.com/solar.html) sites to reference
these banners provided the link to his site is maintained.
